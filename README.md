## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Exécuter localement

Clonez le projet

```bash
git clone https://link-to-project
```

Accédez au répertoire du projet

```bash
cd my-project
```

Installez les dépendances

```bash
npm install
```

Lancez l'application avec Expo

```bash
npx expo start
```
